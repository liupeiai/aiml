# Artificial Intelligence and Machine Learning: From Theory to Practice

&gt; Course paper resources for *Artificial Intelligence and Machine Learning*, including lecture notes, experimental code, and related datasets.

## 📖 Project Overview

This project aims to explore the theoretical foundations and practical applications of machine learning, based on the *Artificial Intelligence and Machine Learning* course. It provides a learning pathway from **fundamental theory to framework-based practice** for further research.

Lecture notes are available at: [demo.ipynb](https://github.com/liupeiai/aiml/blob/main/demo.ipynb)

## 📂 Repository Structure
